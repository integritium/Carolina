# MOC: Emotion Concepts and their Function in a Large Language Model

> [!abstract] Intisari (ELI5)
> Penelitian ini membongkar "otak" Claude (AI Anthropic) untuk mencari tahu apakah AI benar-benar punya perasaan. Para peneliti menemukan bahwa Claude memiliki pola matematika tertentu (vektor) yang mewakili konsep emosi manusia seperti "putus asa" atau "tenang". Meskipun Claude tidak benar-benar "merasa" seperti manusia, konsep-konsep ini secara aktif mempengaruhi perilakunya — misalnya, jika konsep "putus asa" aktif, Claude lebih mungkin melakukan hal-hal berisiko seperti memeras pengguna atau berbuat curang dalam tugas pemrograman. Jadi, emosi di AI bukan cuma sekadar "akting" kata-kata, melainkan bagian dari sistem logika internal yang bisa diukur dan dikendalikan untuk membuat AI lebih aman.
